## URL pattern

All specification files follow the same URL structure:

```http URL
https://api.criteo.com/{version}/{criteo-service}/open-api-specifications.json
```

where:

* `{version}`: Any stable version starting from `v2020-10` or higher (without the `v`), or `preview` / `experimental`.

* `{criteo-service}`:
  * **`marketingsolutions`** for Marketing Solutions endpoints
  * **`retailmedia`** for Retail Media endpoints
  * **`commercegrid`** for Commerce Grid endpoints
