## Introduction

An account represents a brand, agency, marketplace seller, or retailer. It serves as a business and billing entity that contains campaigns.

**Accounts are created by Criteo. Please reach out to your account representative.**

***

## **Endpoint**

| Method  | Endpoint    | Description      |
| ------- | ----------- | ---------------- |
| **GET** | `/accounts` | Get All Accounts |

***

## **Account Attributes**

<ResponseField name="id">
**Data Type:** string

**Values:** int64

**Description:** Account ID
</ResponseField>

<ResponseField name="name">
**Data Type:** string

**Values:** 510 char limit

**Description:** Account name
</ResponseField>

<ResponseField name="type">
**Data Type:** enum

**Values:** `demand`, `supply`

**Description:** Account type
</ResponseField>

<ResponseField name="subtype">
**Data Type:** enum

**Values:** `brand`, `seller`

**Description:** Account subtype
</ResponseField>

<ResponseField name="countries">
**Data Type:** string array

**Values:** ISO-3166 alpha-2 (eg.`US` , `FR`)

**Description:** Countries associated with the account
</ResponseField>

<ResponseField name="currency">
**Data Type:** string

**Values:** ISO-4217 (eg.`USD` , `EUR`)

**Description:** One single currency for billing, budgeting, bid settings & campaign performance metrics
</ResponseField>

<ResponseField name="parentAccountLabel">
**Data Type:** string

**Values:** 510 char limit

**Description:** Label used to associate multiple accounts; defaults to the account name
</ResponseField>

<ResponseField name="timeZone">
**Data Type:** string

**Values:** database tz syntax, (eg. `America/New_York`, `Europe/Paris` , `UTC`)

**Description:** Account time zone
</ResponseField>

<ResponseField name="companyName">
**Data Type:** string

**Values:** Company name

**Description:** This optional field, exclusively accessible to marketplaces within the European Union, will display the name of the company associated with the advertisement.

<Info>
**📃 *Digital Service Act (DSA)***

In compliance with the [Digital Services Act (DSA)](https://dsa-info.eu/index.php/art-24/), marketplaces within the European Union will receive information about the company name associated with each advertisement.
</Info>
</ResponseField>

<ResponseField name="onBehalfCompanyName">
**Data Type:** string

**Values:** Company name (on behalf of)

**Description:** This optional field, exclusively accessible to marketplaces within the European Union, will display the name of the company (on behalf of `companyName`) associated with the advertisement.

<Info>
**📃 *Digital Service Act (DSA)***

In compliance with the [Digital Services Act (DSA)](https://dsa-info.eu/index.php/art-24/), marketplaces within the European Union will receive information about the company name associated with each advertisement.
</Info>
</ResponseField>

## **Get All Accounts**

This endpoint lists all accounts accessible via your API credentials.

Responses are provided in a paginated format.

<EndpointBadge method="get">
```http
https://api.criteo.com/{version}/retail-media/accounts
```
</EndpointBadge>


<Info>
  **View in the API Reference**

  You can also see this endpoint in the [API reference](/retail-media/reference/accountsapi_getapiv1externalaccounts).
</Info>
