* This endpoint and its underlying capabilities only apply to Onsite Display Line items.
* Add-to-basket targeting allows restricting Onsite Display line items to serve only users who have added specific products or products from specific categories to their basket

When using this feature, the PUT endpoint must be called first to initialize it, defining its `scope`. POST endpoints can only alter the list of products or categories.

Either Product IDs or Category IDs must be provided. Providing both simultaneously is not allowed.

## Store Attributes

<ParamField path="enum" type="enum" required>
  *Required on PUT endpoint.*

  Whether listed entries should be included or excluded from the line item

  **Values:** `include`,`exclude`
</ParamField>

<ParamField path="productIds" type="string array">
  List of all Products IDs

  **Values:** -
</ParamField>

<ParamField path="categoryIds" type="string array">
  List of all Category IDs

  **Values:** -
</ParamField>

## List all targeted Stores on an Onsite Display Line Item

<Badge color="blue" size="sm" shape="pill">GET</Badge>

```text
https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/add-to-basket
```

**Sample Request**

```bash
curl -X GET "https://api.criteo.com/{version}/retail-media/preferred-line-items/2465695028166499188/targeting/add-to-basket" \
    -H "Authorization: Bearer myaccesstoken"
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAddToBasketTarget",
        "attributes": {
            "scope": "include",
            "productIds": null,
            "categoryIds": [
                "3922691",
                "4386545",
                "4386646"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

## Initiate targeted Stores list and scope on a Line item

* This endpoint initializes the feature on an Onsite Display Line item.
* If targeted Stores were already defined, those will be overridden.

<Badge color="orange" size="sm" shape="pill">PUT</Badge>

```text
https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/stores
```

**Sample Request**

```bash
curl -X PUT "https://api.criteo.com/{version}/retail-media/preferred-line-items/2465695028166499188/targeting/stores" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAddToBasketTarget",
                "attributes": {
                    "scope": "include",
                    "productIds": null,
                    "categoryIds": [
                        "3922691",
                        "4386545",
                        "4386646"
                    ]
                }
            }
       }'
```

**Sample Response**

```bash
{
    "data": {
        "type": "RetailMediaAddToBasketTarget",
        "attributes": {
            "scope": "include",
            "productIds": null,
            "categoryIds": [
                "3922691",
                "4386545",
                "4386646"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

## Add targeted Product or Category IDs on a Line item

* This endpoint adds target product or category IDs to an Onsite Display Line item.  

<Badge color="red" size="sm" shape="pill">POST</Badge>

```text
https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/add-to-basket/append
```

**Sample Request**

```bash
curl -X POST "https://api.criteo.com/{version}/retail-media/preferred-line-items/2465695028166499188/targeting/add-to-basket/append" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAddToBasketTarget",
                "attributes": {
                    "productIds": null,
                    "categoryIds": [
                        "4380644",
                        "4380645"
                    ]
                }
            }
        }'
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAddToBasketTarget",
        "attributes": {
            "scope": "include",
            "productIds": null,
            "categoryIds": [
                "4380646",
                "4380645",
                "4380644"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

## Remove targeted Product or Category IDs on a Line item

* This endpoint removes previously targeted Add-to-basket categories or products

<Badge color="red" size="sm" shape="pill">POST</Badge>

```text
https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/add-to-basket/delete
```

**Sample Request**

```bash
curl -X POST "https://api.criteo.com/{version}/retail-media/preferred-line-items/2465695028166499188/targeting/add-to-basket/delete" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAddToBasketTarget",
                "attributes": {
                    "categoryIds": [
                        "4380646"
                    ]
                }
            }
        }'
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAddToBasketTarget",
        "attributes": {
            "scope": "include",
            "productIds": null,
            "categoryIds": [
                "4380644",
                "4380645"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```
