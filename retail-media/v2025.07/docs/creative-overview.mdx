* Creative builder allows setting the template variables to build the creative

* Fields required to construct the creative will be based on separate template variables required for individual creative template types

* This doc refers to "template variables" as the individual creative template fields available for each template. 

## Template & Template Variable Model

*TemplateVariable* (array)\
Corresponds to an array of information that can be used as a reference to make a single field in the creatives builder. The model will provide the key/id and specifications on the values that can be set. 

* The TemplateVariable model is present in the **GET** template method

## Template Variable Value

*TemplateVariableValue* (array)\
corresponds to the information that has to be entered into the fields for **POST** and **PUT** creative creation and update calls. Sequentially, to a key/value pair in the final creative settings. The model will attempt to further inform and constrain valid input.

* Fields that are available for setup under the TemplateVariableValue are detailed by the **GET** template method.
