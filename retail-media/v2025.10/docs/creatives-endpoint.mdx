## **Creative Attributes**

<ParamField path="id" type="string">
  **Value:** int64

  Creative ID
</ParamField>

<ParamField path="name" type="string">
  **Value:** 100 char limit

  Creative name
</ParamField>

<ParamField path="status" type="enum">
  **Value:** `Ready`, `In_Use`, `Archived`, `Deleted`

  Creative status
</ParamField>

<ParamField path="brandId" type="int">
  **Value:** int64

  Brand the creative is associated to
</ParamField>

<ParamField path="retailerId" type="int">
  **Value:** int32

  Retailer the creative is designed for
</ParamField>

<ParamField path="associatedLineItemIds" type="string array">
  **Value:** int64

  Line items the creative is setup on
</ParamField>

 

## **Get All Creatives**

This endpoint lists all retailers that carry the brands associated with an account.  Results are not paginated.

<Badge color="blue" size="sm" shape="pill">GET</Badge>

```http
https://api.criteo.com/{version}/retail-media/accounts/{accountId}/creatives
```
