<Note>
  ### Getting Started

  An _audience_ represents an aggregation of all users who interacted with specific brands or categories over a defined period. Audiences are used only with preferred deals line items with the audience type options of delivering audiences based on a **customerList** or **userBehavior**. Additionally, in Audience V2, there is now an option to specify the user segments to _"include"_ or _"exclude"_ in your user segment targeting.
</Note>

## **Audience Attributes**

<ParamField path="id" type="string">
  Audience ID

  **Values:** int64
</ParamField>

<ParamField path="type" type="enum" required>
  Type of the audience

  **Values:** `customerList`, `userBehaviour`
</ParamField>

<ParamField path="name" type="string" required>
  Audience name

  **Values:** -
</ParamField>

<ParamField path="userAction" type="enum" required>
  User behavior that is targeted

  **Values:** `viewed`, `buy`, `addToCart`
</ParamField>

<ParamField path="lookbackWindow" type="enum" required>
  Length of the loopback window, in days

  **Values:** `P7D`, `P14D`, `P30D`, `P45D`, `P60D`, `P90D`, `P120D`, `P150D`, `P180D`
</ParamField>

<ParamField path="brandIds" type="int array">
  *Optional.*

  Brand ids to target

  - **Note:** Either categoryId or brandIds must be specified in the call when creating new audiences

  **Values:** list of brand ids
</ParamField>

<ParamField path="categoryIds" type="int array">
  *Optional.*

  Category ids to target.

  - **Note:** Either categoryId or brandIds must be specified in the call when creating new audiences

  **Values:** list of category ids
</ParamField>

<ParamField path="retailerId" type="string" required>
  Id of the retailer associated with the audience

  **Values:** -
</ParamField>

<ParamField path="userBehaviorDetails" type="object">
  Details related to the user behavior audience.

  **Values:** UserBehaviorDetails
</ParamField>

<ParamField path="inclusiveSegment" type="object" required>
  Details related to inclusive user behavior audience

  **Values:** UserBehaviorAudienceSegment
</ParamField>

<ParamField path="exclusiveSegment" type="object">
  *Optional.*

  Details related to exclusive user behavior audience. Can only be specified if inclusiveSegment is specified

  **Values:** UserBehaviorAudienceSegment
</ParamField>

<ParamField path="minPrice" type="double">
  *Optional.*

  The min price of targeted skus.

  **Values:** -
</ParamField>

<ParamField path="minPrice" type="double">
  *Optional.*

  The max price of targeted skus.

  **Values:** -
</ParamField>

<ParamField path="customerListDetails" type="object">
  *Optional.*

  Details related to the customer list audience

  **Values:** customerListDetails
</ParamField>

<ParamField path="size" type="string" required>
  Number of users in the customer list audience.

  **Values:** int64
</ParamField>

## **Get All Audiences by Account Id**

<Badge color="blue" size="sm" shape="pill">GET</Badge>

```http
https://api.criteo.com/{version}/retail-media/v2/accounts/{accountId}/audiences
```

**Sample Request**

<CodeGroup>

```bash cURL
curl -L -X GET 'https://api.criteo.com/{version}/retail-media/v2/accounts/5/audiences?pageSize=3' \
-H 'Accept: application/json' \
-H 'Authorization: Bearer <MY_ACCESS_TOKEN>'
```

```python Python
import http.client

conn = http.client.HTTPSConnection("api.criteo.com")
payload = ''
headers = {
  'Accept': 'application/json',
  'Authorization': 'Bearer <MY_ACCESS_TOKEN>'
}
conn.request("GET", "/{version}/retail-media/v2/accounts/5/audiences?pageSize=3", payload, headers)
res = conn.getresponse()
data = res.read()
print(data.decode("utf-8"))
```

