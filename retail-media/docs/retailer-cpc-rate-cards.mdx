<Note>
  ### Getting Started

  \-Retailers have bid floors (minimum cpc) that determine the minimum allowed values that line items can bid within their inventory\
  -Minimum CPC exists at the retailer and [Category ](/retail-media/docs/categories) level\
  -Target Bids for [Open Auction Line Items](/retail-media/docs/oa-line-items) creation require a bid equal to or greater than the retailer's minimum CPC
</Note>

## **Get Retailer Minimum CPC Rate**

<Badge color="blue" size="sm" shape="pill">GET</Badge>

```text
https://api.criteo.com/{version}/retail-media/retailers/{retailerId}/retailer-cpc-rates
```

**Sample Request**

```bash
# Request
 curl -L -X GET 'https://api.criteo.com/{version}/retail-media/retailers/299/retailer-cpc-rates' \
-H 'Authorization: Bearer <MY_ACCESS_TOKEN'
```

**Sample Response**

```json
{
    "data": {
        "attributes": {
            "retailerId": "299",
            "minimumBid": 0.2000
        },
        "id": "299",
        "type": "RetailerCpcRateCard"
    },
    "warnings": [],
    "errors": []
}
```

## Get Retailer Minimum CPC Rate for Each Category

<Badge color="blue" size="sm" shape="pill">GET</Badge>

```text
https://api.criteo.com/{version}/retail-media/retailers/{retailerId}/category-cpc-rates
```

**Sample Request**

```bash
//Response
curl -L -X GET 'https://api.criteo.com/{version}/retail-media/retailers/299/category-cpc-rates' \
-H 'Authorization: Bearer <MY_ACCESS_TOKEN>'
```

**Sample Response**

```json
{
    "data": [
        {
            "attributes": {
```
