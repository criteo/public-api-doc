* This endpoint and its underlying capabilities only apply to Onsite Display Line items.
* Audience targeting allows you to restrict Onsite Display line items to serve only users belonging to some predefined audience.

When using the feature, the PUT endpoint must be used to first initialize it, defining its `scope`. POST endpoints can only alter the list of audiences.

## Audience Attributes

<ParamField path="scope" type="enum" required>
  *Required on PUT endpoint.*

  Whether listed audiences should be included or excluded from the line item

  **Values:** `include`,`exclude`
</ParamField>

<ParamField path="audienceIds" type="string array">
  List of all audiences

  **Values:** -
</ParamField>

## List all targeted Audiences on an Onsite Display Line Item

<Badge color="blue" size="sm" shape="pill">GET</Badge>

```text
https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/audiences
```

**Sample Request**

```bash
curl -X GET "https://api.criteo.com/{version}/retail-media/preferred-line-items/2465695028166499188/targeting/audiences" \
    -H "Authorization: Bearer myaccesstoken"
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAudienceTarget",
        "attributes": {
            "scope": "include",
            "audienceIds": [
                "266154384519348224",
                "266154384519348225"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

## Initiate targeted Audiences list and scope on a Line item

* This endpoint initializes the feature on an Onsite Display Line item.
* If targeted Audiences were already defined, those will be overridden

<Badge color="orange" size="sm" shape="pill">PUT</Badge>

```text
https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/audiences
```

**Sample Request**

```bash
curl -X PUT "https://api.criteo.com/{version}/retail-media/preferred-line-items/2465695028166499188/targeting/audiences" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAudienceTarget",
                "attributes": {
                    "scope": "exclude",
                    "audienceIds": [
                        "266154384519348224",
                        "266154384519348225"
                    ]
                }
            }
        }'
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAudienceTarget",
        "attributes": {
            "scope": "exclude",
            "audienceIds": [
                "266154384519348224",
                "266154384519348225"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

## Add targeted Audiences on a Line item

* This endpoint adds target audiences to an Onsite Display Line item.  

<Badge color="red" size="sm" shape="pill">POST</Badge>

```text
https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/audiences/append
```

**Sample Request**

```bash
curl -X POST "https://api.criteo.com/{version}/retail-media/preferred-line-items/2465695028166499188/targeting/audiences/append" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAudienceTarget",
                "attributes": {
                    "audienceIds": [
                        "266154384519348226"
                    ]
                }
            }
        }'
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAudienceTarget",
        "attributes": {
            "scope": "include",
            "audienceIds": [
                "266154384519348224",
                "266154384519348225",
                "266154384519348226"
            ]
        }
    },
    "errors": [],
    "warnings": []
}
```

https://api.criteo.com/{version}/retail-media/preferred-line-items/{line-item-id}/targeting/audiences/delete
