- This endpoint and its underlying capabilities only apply Preferred Deals Line items.
- Audience targeting allows you to restrict a Preferred Deals line items to serve only for users belonging to some predefined audience

# Endpoints

- `GET /preferred-line-items/`{line-item-id}`/targeting/audiences` Get All targeted Audiences on a specific Line Item
- `PUT /preferred-line-items/`{line-item-id}`/targeting/audiences` Initialize or Override Audiences targeting on a specific Line Item
- `POST /preferred-line-items/`{line-item-id}`/targeting/audiences/append` Add targeted Audiences to a Specific Line Item
- `POST /preferred-line-items/`{line-item-id}`/targeting/audiences/delete` Remove targeted Audiences from a Specific Line Item

When using the feature, the PUT endpoint must be used to first initialize it, defining its `scope`. POST endpoints can only alter the list of audiences.

# List all targeted Audiences on a Preferred Deals Line Item

<Badge color="blue" size="sm" shape="pill">GET</Badge>

```http
https://api.criteo.com/preview/retail-media/preferred-line-items/`{line-item-id}`/targeting/audiences
```

**Sample Request**

```bash
curl -X GET "https://api.criteo.com/preview/retail-media/preferred-line-items/2465695028166499188/targeting/audiences" \
    -H "Authorization: Bearer myaccesstoken"
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAudienceTarget",
        "attributes": {
            "scope": "include",
            "audienceIds": [
                "266154384519348224",
                "266154384519348225"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

# Initiate targeted Audiences list and scope on a Line item

- This endpoint initializes the feature on a Preferred Deals Line item.
- If targeted Audiences were already defined, those will be overridden

<Badge color="orange" size="sm" shape="pill">PUT</Badge>

```http
https://api.criteo.com/preview/retail-media/preferred-line-items/`{line-item-id}`/targeting/audiences
```

**Sample Request**

```bash
curl -X PUT "https://api.criteo.com/preview/retail-media/preferred-line-items/2465695028166499188/targeting/audiences" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAudienceTarget",
                "attributes": {
                    "scope": "exclude",
                    "audienceIds": [
                        "266154384519348224",
                        "266154384519348225"
                    ]
                }
            }
        }'
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAudienceTarget",
        "attributes": {
            "scope": "exclude",
            "audienceIds": [
                "266154384519348224",
                "266154384519348225"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```
