This section will explain you how to edit your App information (before publication).

1. Select an application you have previously created in your My Apps space

![1104](https://files.readme.io/423bba6-myapp1.png "myapp1.png")

2. Click on Edit at the right of the App name

![1174](https://files.readme.io/6f9205b-myapp2.png "myapp2.png")

3. Edit the App name, description or image (possible only before publishing an App)

![1480](https://files.readme.io/de51863-app3.png "app3.png")
