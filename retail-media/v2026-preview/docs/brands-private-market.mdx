<Note>
  ### A FEW THINGS TO KNOW

  1 - A *brand* is a collection of products marketed & sold under the same umbrella name.\
   2 - Brands associated with the account determine the products available for an account to promote on retailer sites.\
  3 - An account may have access to one or more brands, typically managed by Criteo.\
  4 - Brand attributes are standardized across retailers
</Note>

## **Brand Attributes**

#### `id`

- **Data Type:** string
- **Values:** int64

**Description:** Brand ID

#### `name`

- **Data Type:** string
- **Values:** 510 char limit

**Description:** Brand name

## **Get All Brands**

This endpoint lists all brands associated with an account.  Results are paginated.

<Badge color="green" size="sm" shape="pill">GET</Badge>

```text
https://api.criteo.com/preview/retail-media/accounts/{private-market-accountId}/brands
```

**Sample Request** 

*In the example, the API calls for all the brands that are mapped to the private market account ID`433350412678656000`*

<CodeGroup>

```bash cURL
curl -L 'https://api.criteo.com/preview/retail-media/accounts/433350412678656000/brands' \
-H 'Accept: application/json' \
-H 'Authorization: Bearer <MY_ACCESS_TOKEN>'
```
