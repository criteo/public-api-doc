## Overview

**Retail Media Insights** is a dashboard section located in the Demand side of Commerce Max.

This asynchronous API reflects the **Share of Voice by Category** and the **Share of Voice by Keyword** dashboards in the Insights section of Retail Media Analytics.

The expected user is a Demand/Supply Side user accessing via a Commerce Max network account.

This insight report highlights traffic (impressions and click share) with 2 different granularity levels:

* Category
* Keyword

The granularity level is selectable in the query.

<Info>
  You can find the Insight Share of Voice endpoint in the API Reference [here](/retail-media/v2026-preview/reference/analytics/generate-share-of-voice-insight).
</Info>

### Request Response

Initiates an asynchronous export of the Share of Voice by keyword report.

#### Response 200 - Success

When the response is successful, the GUID ID used to retrieve the report is included in the response body. The user needs to use this ID to fetch their report by using the following endpoint:

`https://api.criteo.com/{version}/retail-media/insights/{reportId}/output`

Report status can also be queried via:

`https://api.criteo.com/{version}/retail-media/insights/{reportId}/status`

{
  "data": {
