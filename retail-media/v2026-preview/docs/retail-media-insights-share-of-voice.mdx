## Introduction

Retail Media Insights exposes performance dashboards available on the Demand & Supply Sides of Commerce Max.

The **Share of Voice** API is an asynchronous reporting endpoint reflecting the _Share of Voice_ dashboard available in Retail Media Insights. It allows users to retrieve impression and click share metrics at a **served category and keyword levels**.

The primary purpose of this endpoint is to fetch impression and click share metrics across categories where ads were served on retailer sites.

<Info>
  **Asynchronous Reports**

  This endpoint initiates an asynchronous export.  
  The response returns a report ID that must be used to:

  * Check report generation status
  * Download the generated report once ready
</Info>
