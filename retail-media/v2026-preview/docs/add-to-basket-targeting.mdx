- This endpoint and its underlying capabilities only apply Preferred Deals Line items.
- Add-to-basket targeting allows to restrict a Preferred Deals line items to serve only for users who have added specific products, or products from specific categories, to their basket

# Endpoints

- `GET /preferred-line-items/`{line-item-id}`/targeting/add-to-basket` Get All targeted Add-to-basket products or categories on a specific Line Item
- `PUT /preferred-line-items/`{line-item-id}`/targeting/add-to-basket` Initialize or Override add-to-basket targeting on a specific Line Item
- `POST /preferred-line-items/`{line-item-id}`/targeting/add-to-basket/append` Add targeted Add-to-basket products or categories to a Specific Line Item
- `POST /preferred-line-items/`{line-item-id}`/targeting/add-to-basket/delete` Remove targeted Add-to-basket products or categories from a Specific Line Item

When using the feature, the PUT endpoint must be used to first initialize it, defining its `scope`. POST endpoints can only alter the list of product or categories.

Either Product IDs or Category IDs must be provided. Providing both simultaneously is not allowed.

# List all targeted Stores on a Preferred Deals Line Item

<EndpointBadge method="get">
```http
https://api.criteo.com/preview/retail-media/preferred-line-items/`{line-item-id}`/targeting/add-to-basket
```
</EndpointBadge>

**Sample Request**

```bash
curl -X GET "https://api.criteo.com/preview/retail-media/preferred-line-items/2465695028166499188/targeting/add-to-basket" \
    -H "Authorization: Bearer myaccesstoken"
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAddToBasketTarget",
        "attributes": {
            "scope": "include",
            "productIds": null,
            "categoryIds": [
                "3922691",
                "4386545",
                "4386646"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

# Initiate targeted Stores list and scope on a Line item

- This endpoint initializes the feature on a Preferred Deals Line item.
- If targeted Stores were already defined, those will be overridden

<EndpointBadge method="put">
```http
https://api.criteo.com/preview/retail-media/preferred-line-items/`{line-item-id}`/targeting/stores
```
</EndpointBadge>

**Sample Request**

```bash
curl -X PUT "https://api.criteo.com/preview/retail-media/preferred-line-items/2465695028166499188/targeting/stores" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAddToBasketTarget",
                "attributes": {
                    "scope": "include",
                    "productIds": null,
                    "categoryIds": [
                        "3922691",
                        "4386545",
                        "4386646"
                    ]
                }
            }
       }'
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAddToBasketTarget",
        "attributes": {
            "scope": "include",
            "productIds": null,
            "categoryIds": [
                "3922691",
                "4386545",
                "4386646"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```

# Add targeted Product or Category IDs on a Line item

- This endpoint adds target product or category IDs to a Preferred Deals Line item.  

<EndpointBadge method="post">
```http
https://api.criteo.com/preview/retail-media/preferred-line-items/`{line-item-id}`/targeting/add-to-basket/append
```
</EndpointBadge>

**Sample Request**

```bash
curl -X POST "https://api.criteo.com/preview/retail-media/preferred-line-items/2465695028166499188/targeting/add-to-basket/append" \
    -H "Authorization: Bearer myaccesstoken" \
    -H "Content-Type: application/json" \
    -d '{
            "data": {
                "type": "RetailMediaAddToBasketTarget",
                "attributes": {
                    "productIds": null,
                    "categoryIds": [
                        "4380644",
                        "4380645"
                    ]
                }
            }
        }'
```

**Sample Response**

```json
{
    "data": {
        "type": "RetailMediaAddToBasketTarget",
        "attributes": {
            "scope": "include",
            "productIds": null,
            "categoryIds": [
                "4380646",
                "4380645",
                "4380644"
            ]
        }
    },
    "warnings": [],
    "errors": []
}
```
