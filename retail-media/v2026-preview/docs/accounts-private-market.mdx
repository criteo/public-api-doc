<Note>
  ### A FEW THINGS TO KNOW

  An account represents a brand, agency, marketplace seller, or retailer; it is a business & billing entity that contains campaigns. 

  Your account is the supply account in the private market, which hosts all the individual private market demand accounts. Each private market demand account belongs to a brand buying within the retailer's inventory.

  **Private market demand accounts are created by Criteo; please reach out to your account representative.**
</Note>

<Danger>
  **X-Origin-Account Header**

  Pass the `X-Origin-Account` header to specify which private market (brand) account your API call should return results for.

  * If your application doesn't have access to the accountID that is passed in the headers, your api call will return a 403 Forbidden response. If a correct account ID that your application has access to is passed, then a 200 response
  * Endpoints where `accountId` is passed in the path takes priority, so headers won't affect the api call
  * **Note:** If your api call doesn't specify this header, the call will still work returning a 200 response
</Danger>

## **Account Attributes**

<ParamField path="id" type="string">
  The private market (Brand) account ID

  **Values:** int64
</ParamField>

<ParamField path="name" type="string">
  The private market (Brand) account name

  **Values:** 510 char limit
</ParamField>

<ParamField path="type" type="enum">
  Account type

  **Values:** `demand`, `supply`
</ParamField>

<ParamField path="subtype" type="enum">
  Account subtype

  **Values:** `brand`, `seller`
</ParamField>

<ParamField path="countries" type="string array">
  Countries associated with the account

  **Values:** ISO-3166 alpha-2 (eg.`US` , `FR`)
</ParamField>

<ParamField path="currency" type="string">
  One single currency for billing, budgeting, bid settings & campaign performance metrics

  **Values:** ISO-4217 (eg.`USD` , `EUR`)
</ParamField>

<ParamField path="parentAccountLabel" type="string">
  Label used to associate multiple accounts; defaults to the account name

  **Values:** 510 char limit
</ParamField>

<ParamField path="timeZone" type="string">
  Retailer time zone

  **Values:** database tz syntax, (eg. `America/New_York`, `Europe/Paris` , `UTC`)
</ParamField>

## **Get All Accounts**

This endpoint lists all accounts that your API credentials have access to.  Response results will be provided in paginated form

<EndpointBadge method="get">
```http
https://api.criteo.com/preview/retail-media/accounts
```
</EndpointBadge>
