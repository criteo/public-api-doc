## Introduction

This guide documents the new `DELETE Audience` endpoint, which enables clients to delete multiple audiences by ID in a single request.

***

## Endpoint

<table>
  <thead>
    <tr>
      <th>Verb</th>
      <th>Endpoint</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>POST</strong></td>
      <td><code>https://api.criteo.com/preview/retail-media/accounts/audiences/delete</code></td>
      <td>Delete multiple audiences by ID</td>
    </tr>
  </tbody>
</table>

<Info>
  * This endpoint uses the `POST` method to perform a **bulk delete**.* Deletions are **irreversible**. Use with care.* Ensure the API client has the necessary permissions and is properly authenticated.
</Info>

***

## Delete Audiences by ID

<Badge color="red" size="sm" shape="pill">POST</Badge>

```text
POST https://api.criteo.com/preview/retail-media/accounts/audiences/delete \
```

**Sample Request**

```json
{
  "data": [
    {
      "attributes": {},
      "id": "string",
      "type": "string"
    }
  ]
}
```

**Sample Response (200)**

```json
{
  "data": [
    {
      "attributes": {},
      "id": "string",
      "type": "string"
    }
  ]
}
```

**Sample Response (204)**

```text
(no content)
```
