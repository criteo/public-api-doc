## Specifications

The format includes an image with a redirection.\
Depending on the placement width we will show a different image.

### Technical details

The Impulse Branding format has three different images that will show depending on the placement width.\
This is to offer more flexibility with the different breakpoints we might have on the website where the format is shown.

There is a safe zone centered within the image on the two biggest images (see below in the "Assets size" section).\
This area will remain visible for every breakpoint, which means all important content must be placed there when creating the image.\
Depending on placement size, these image can be cropped on both sides (height stays the same) :

### Placement specificities

The format can be placed as a “sticky” unit.\
This means the format will stay at the bottom of the viewport, independently of the scrolling of the page.

![1716](https://files.readme.io/8e88f15-Frame-27.jpg "Frame-27.jpg")

For this specific type of placement only, and to ensure a good user experience and legibility, the unit should have : 

* a drop shadow on the image container. See below an exemple of CSS shadow property that can be use :

```css
filter: drop-shadow(0px -4px 10px rgba(0, 0, 0, 0.1));
```

* a "close" icon, that once clicked enables user to remove the format.

# Tracking ad activity

Ad activity is tracked through beacons. Those are GET HTTPS calls on the client side that need to be triggered both when a set of standard actions take place, as well as when specific actions take place 

The list of available beacons will depend on the format of the ad.  

If the user repeats the same action multiple times (e.g. increasing the quantity of a given product in his basket), the corresponding beacon needs to be fired the same number of times. 

Below are the beacons to be triggered for the Display Panel format:
