## Specifications

The Impulse Product format has 2 different images that will show depending on the placement width.\
Below a certain width, there are no images anymore for a better readability.

On the two images, there is a safe zone on the right side of these images (see below in the "Assets size" section).\
This area will remain visible for every breakpoint, which means all important content must be placed there when creating the images.

Depending on the width of the placement, these image can be cut on the left side.

#### No image

When for format placement is smaller than 768px in width, no image is shown anymore and the text area is displayed on top of a background colour. This colour is the same used on both sides when the format placement is bigger than 1850px.\
The text area and and product box will have the same width (each part takes 50% of the width of the placement).

#### No image - mobile version

When the format placement is smaller than 460px in width, the unit will keep an aspect ratio of 5,75/2 for the width & height (e.g. 460x160px for a placement of 460px).

# Tracking Ad Activity

Ad activity is tracked through beacons. These are GET HTTPS calls on the client side that need to be triggered both when a set of standard actions take place and when specific actions take place.

The list of available beacons will depend on the format of the ad.  

Some beacons are tied to the whole ad, while others are tied to specific products. Note that any user interaction with the ad should only trigger one call.\
If the user repeats the same action multiple times (for example, increasing the quantity of a given product in his basket), the corresponding beacon needs to be fired the same number of times.
