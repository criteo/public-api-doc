# Introduction

This page provides an overview of various digital ad formats available in the Criteo Retail Media solution.

Each format  is tailored to meet specific advertising goals, (maximizing visibility, increasing user engagement, or seamlessly integrating ads into a platform's user experience).

There are three main categories of formats.

* **Onsite Display** including:
  * Commerce Display Grid Duet,
  * Commerce Display Spotlight,
  * Commerce Display Grid Shelf,
  * Branding Display Grid Solo,
  * Branding Display Spotlight Solo.
* The **Sponsored Products** format,
* The **Video formats**, including:
  * The Commerce Video Spotlight format,
  * The Branding Video Standout format,
  * The Commerce Video Grid Duet format.

***

# Onsite Display Formats

<Warning>
  As of November 2025, Criteo has redesigned and renamed **five formats for Onsite Display**. Those formats are simpler to implement and fit better with current standards and technical constraints. We strongly recommended using the new formats for any new implementation, as legacy formats will be progressively phased out.
</Warning>

## What changed?

The five following formats have been redesigned and renamed: `Flagship`, `Showcase`, `Butterfly`, `Digital Shelf Talker` and `Display Panel`.

The mobile-specific formats `Branded Header` and `Interactive Header` were integrated into the new formats.

`The`IAB Banners` remains unchanged.
