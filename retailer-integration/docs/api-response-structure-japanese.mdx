## レスポンス要素

APIレスポンスの構造は全てのフォーマットにおいて共通です。詳細については以下をご覧ください。

### `Status`

タイプ：`string`

値：`OK`（レスポンス成功の場合）またはエラータイプ。

エラーが発生した場合、該当するエラーメッセージのリストを含む配列がレスポンスに追加されます（例：`Missing required parameter x`）。

***

### `Placements`

タイプ：`array of placement objects`

各広告掲載枠名は、`viewCategory_API_desktop-Video` のようにキーとして使用されます。

各キーの値は広告の配列を示します。 各広告には以下のプロパティがあります。

* **`format`**：返される形式を示します。

* **`products`**：広告フォーマットに含まれる SKU のリストと、各 SKU に関する追加のメタデータ（画像、製品 URL、親 SKU ID など）<br/>

* **`products_order`**：以下のような SKU の在庫状況に応じて、リアルタイムの代替製品が許可されます。<br/>

  * **`products`**：製品 ID の配列。 Criteo によって返される商品の順序は、SKU が入手不可能または在庫切れでない限り、サイトに表示される SKU の順序と一致する必要があります。<br/>
  * **`isMandatory`**: true の場合、この商品が利用できないときは広告ユニット全体を**表示しないでください。**。 false の場合、広告は表示できますが、ボタン/製品タイルはユニットから削除する必要があります。

***

### `Rendering`

* **`rendering`**：画像アセット、色、カスタムラベルなど、広告をレンダリングするときに使用するフォーマット固有のプロパティ。

<Warning>
  レンダリングセクション内のパラメータを**動画フォーマットのレンダリングに使用しないでください**。 詳細については、[下の表](/retailer-integration/docs/api-response-structure#rendering-fields-japanese)を参照してください。

  弊社は、将来必要に応じてこれらの値を削除する権利を保有します。
</Warning>

<Info>
  **動画のフォーマット**

  レンダリング機能の詳細については、[フォーマットの仕様](/retailer-integration/docs/format-overview-japanese)（広告レンダリングセクション）を参照してください。 このセクションには、[Commerce Video Spotlight の仕様](/retailer-integration/docs/commerce-video-spotlight-japanese)も記載されています。
</Info>

下記の `TagVideoVAST ` フィールドを参照してください。

***

### `TAGVideoVAST`

* **`TagVideoVAST`** または **`XmlVideoVAST`：**
  * **`TagVideoVAST`** *（デフォルト）* **：** VAST をサポートするビデオ プレーヤー経由で VAST XML を読み込むための URL。
  * **`XmlVideoVAST`** *（小売業者が VAST 定義を取得するために追加の API 呼び出しを行わない場合）*：VAST 標準をサポートするビデオ プレーヤーを提供するための VAST XML。

* **`OnViewBeacon`、`OnLoadBeacon`、`OnClickBeacon` など**：特定のアクションが発生したときに Criteo にトリガー/送信されるビーコン。 詳細については、セクションを参照してください。

***

## フィールドのレンダリング

以下は、API レスポンス内のレンダリングフィールドの例です。

```json
{
  "rendering": {
    "background_video": "{\n  \"url\": \"https://static.criteo.net/design/dt/commerce_max/retailer_06/d97eb13b8e3d496eaddef7b4c998d286_video.mp4\",\n  \"width\": 720,\n  \"height\": 1280,\n  \"duration\": \"00:00:06\"\n}",
    "video_caption_file": "https://static.criteo.net/video/onsite_cc/laptop_vertical.vtt",
    "video_alt_text": "Laptop",
    "border_color": "#712329",
    "video_optional_redirect_url": "https://www.retailer.com/shop/search-results.html?q=laptop",
    "video_optional_redirect_url_app": "",
    "video_media_files": "[\"background_video\"]"
  }
}
```

以下は、ページ内のフォーマットのレンダリングに関する API レスポンスフィールドの説明です。

<Warning>
  これらのフィールドは、動画フォーマットに応じて変更される可能性があります。
</Warning>
