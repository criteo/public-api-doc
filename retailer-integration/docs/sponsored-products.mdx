# Introduction

Sponsored products are native ads that appear directly within a retailer’s product listings or search results, blending seamlessly with the site's content.

***

# Response description

The response of the API call will give you all the information needed to render the ads and track them.

For this reason, the important data are the `productId` in the `products` list to know which product should be displayed, and the different beacons to be able to measure ad performance.

You can find more information about the beacons in the next [chapter](/retailer-integration/docs/retail-media-beacons) and a list of all possible attributes for Sponsored products below.

***

</table>

## Formats

This object contains only two members.

The `format` is the type of format. For Sponsored Products, it will always be `sponsored_products`. `products` will then be a list of products that has to be rendered.

***

## Products

<table>
  <thead>
    <tr>
      <th>Attributes</th>
      <th>Format</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>ProductId</code></td>
      <td>String</td>
      <td>Product ID of the product that matches what Criteo has in the product feed</td>
    </tr>
    <tr>
      <td><code>ProductName</code><br /><code>Image</code><br /><code>Rating</code><br /><code>Price</code><br /><code>ComparePrice</code><br /><code>Shipping</code><br /><code>PromoText</code><br /><code>ParentSKU</code></td>
      <td>String</td>
      <td>Values that are in the feed, describing the <code>productId</code> above</td>
    </tr>
    <tr>
      <td><code>RenderingAttributes</code></td>
      <td>String containing a JSON object</td>
      <td>Values in the feed that are not standard, but are needed to render the ad. If you need these, contact your Criteo representative</td>
    </tr>
    <tr>
      <td><code>ClientAdvertiserId</code></td>
      <td>String</td>
      <td>Internal ID</td>
    </tr>
    <tr>
      <td><code>AdvertiserId</code></td>
      <td>String</td>
      <td>Criteo ID of the advertiser who sent the ad</td>
    </tr>
    <tr>
      <td><code>ProductPage</code><br /><code>OnLoadBeacon</code><br /><code>OnViewBeacon</code><br /><code>OnClickBeacon</code><br /><code>OnBasketChangeBeacon</code><br /><code>OnWishlistBeacon</code></td>
      <td>String containing a URL</td>
      <td>More information on the beacons can be found in the next <a href="#tracking-ad-activity">chapter</a></td>
    </tr>
  </tbody>
</table>
