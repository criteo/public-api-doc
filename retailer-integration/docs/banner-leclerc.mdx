### Available Options:

In the options area (only one option is available at the same time per format). 

* Redirection (redirection link)
  * At the button's click, it redirects to the link defined on the platform. It's possible to have it on *Self* and *Blank* redirection options (not applicable for App)
* Shop (redirection to a "boutique")
  * At the button's click, it redirects to the link defined on the platform. It's possible to have it on *Self* and *Blank* redirection options (not applicable for App). In the landing page we find two assets for a banner (desktop and mobile), a title and all information that concerns the products (title, price, beacon...) 

#### Asset Sizes:

* Background image: 1594x200px

* Mobile image: 797x200px

* Shop background image: 1395x70px 

* Shop mobile image: 797x200px

# Tracking Ad Activity

Ad activity is tracked through beacons. These are GET HTTPS calls on the client side that need to be triggered both when a set of standard actions take place and when specific actions take place. 

The list of available beacons depends on the format of the ad.  

Some beacons are tied to the whole ad, while others are tied to specific products. Note that any user interaction with the ad should only trigger one call.\
If the user repeats the same action multiple times (for example, increasing the quantity of a given product in his basket), the corresponding beacon needs to be triggered the same number of times.
