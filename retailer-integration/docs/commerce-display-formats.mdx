Commerce Display Formats  creatives combine both brand assets and dynamic, retailer native real-time SKU information from product feed, which allows Brands and Retailers to tell their story on a Retailer site or app.

In this section, we specify details for the following formats:

* Flagship
* Showcase
* Butterfly
* Branded Header
* Interactive Header
