## 응답 요소

모든 포맷은 아래에 설명된 동일한 API 응답 구조를 가집니다.

### `상태`

유형: `문자열`

값: `OK`(성공적인 응답의 경우) 또는 오류 유형.

오류가 발생한 경우 해당 오류 메시지 목록이 포함된 배열이 응답에 추가됩니다(예: `Missing required parameter x`).

***

### `배치`

유형: `배치 객체의 배열`

각 배치 이름은 키로 사용됩니다(예: `viewCategory_API_desktop-Video`).

각 키의 값은 광고의 배열입니다. 각 광고에는 다음과 같은 속성이 있습니다.

* **`format`**: 반환되는 포맷을 나타냅니다.

* **`products`**: 광고 포맷에 포함되는 SKU 목록과 각 SKU에 대한 추가 메타데이터(이미지, 제품 URL, 상위 SKU ID 등)입니다.<br/>

* **`products_order`**: SKU의 재고 여부에 따라 일부 실시간 제품 대체가 허용됩니다:<br/>

  * **`products`**: 제품 ID의 배열입니다. 크리테오에서 반품되는 제품의 순서는 사이트에 표시된 SKU의 순서와 일치해야 합니다. 단, SKU가 없거나 재고가 없는 경우는 예외입니다.<br/>
  * **`isMandatory`**: 참(true)인 경우, 해당 제품이 없는 경우 전체 광고 단위는 표시되면 **않됩니다**. 거짓(false)인 경우 광고는 표시될 수 있지만, 버튼/제품 타일은 해당 광고 단위에서 제거되어야 합니다.

***

### `렌더링`

* **`rendering`**: 이미지 애셋, 색상, 사용자 정의 라벨을 포함해 광고를 렌더링할 때 사용하는 포맷별 속성입니다.

<Warning>
  렌더링 섹션의 모든 비디오 파라미터는 **비디오 포맷을 렌더링하는 데 사용해서는 안 됩니다.** 자세한 내용은 [아래 표](/retailer-integration/docs/api-response-structure#rendering-fields)를 참조하세요.

  크리테오는 향후 언제든지 이러한 비디오 값을 삭제할 권리가 있습니다.
</Warning>

<Info>
  **비디오 포맷**

  렌더링 기능에 대한 자세한 내용은 [포맷 사양](/retailer-integration/docs/format-overview)(광고 렌더링 섹션)을 참조하세요. 이 섹션에서 [Commerce Video Spotlight 사양](/retailer-integration/docs/commerce-video-spotlight)도 찾을 수 있습니다.
</Info>

아래 설명된 `TagVideoVAST` 필드를 참조하세요.

***

### `TAGVideoVAST`

* **`TagVideoVAST`**또는 **`XmlVideoVAST`:**
  * **`TagVideoVAST`** *(기본값)* **:** VAST를 지원하는 비디오 플레이어를 통해 VAST XML을 로드하는 URL입니다.
  * **`XmlVideoVAST`** *(유통사가 VAST 정의를 가져오기 위해 추가 API 호출을 하지 않는 경우)*: VAST 표준을 지원하는 비디오 플레이어를 제공하는 VAST XML입니다.

* **`OnViewBeacon`,`OnLoadBeacon`, `OnClickBeacon` 등**: 특정 조치가 취해질 때 크리테오로 트리거/실행되는 비콘입니다. 자세한 내용은 해당 섹션을 참조하세요.

***

## 렌더링 필드

다음은 API 응답에서 필드를 렌더링하는 예입니다.

```json
{
  "rendering": {
    "background_video": "{\n  \"url\": \"https://static.criteo.net/design/dt/commerce_max/retailer_06/d97eb13b8e3d496eaddef7b4c998d286_video.mp4\",\n  \"width\": 720,\n  \"height\": 1280,\n  \"duration\": \"00:00:06\"\n}",
    "video_caption_file": "https://static.criteo.net/video/onsite_cc/laptop_vertical.vtt",
    "video_alt_text": "Laptop",
    "border_color": "#712329",
    "video_optional_redirect_url": "https://www.retailer.com/shop/search-results.html?q=laptop",
    "video_optional_redirect_url_app": "",
    "video_media_files": "[\"background_video\"]"
  }
}
```

아래는 페이지 내에서 포맷을 렌더링하기 위한 API 응답 필드에 대한 설명입니다.

<Warning>
  이러한 필드는 비디오 포맷에 따라 변경될 수 있습니다.
</Warning>

<table>
  <thead>
    <tr>
      <th>필드</th>
      <th>설명</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>video&#95;background</code> (이 필드 내의 모든 정보 포함)</td>
      <td><strong>이 필드는 사용하지 마십시오.</strong> 크리테오 전용입니다. 모든 비디오 관련 정보, 애셋 및 트래킹은 VAST XML에서만 사용해야 합니다.</td>
    </tr>
    <tr>
      <td><code>video&#95;caption&#95;file</code></td>
      <td>비디오 캡션 파일(WebVTT)</td>
    </tr>
    <tr>
      <td><code>video&#95;optional&#95;redirect&#95;url</code> <code>video&#95;optional&#95;redirect&#95;url&#95;app</code></td>
      <td><strong>이 필드는 사용하지 마십시오.</strong> 크리테오 전용입니다</td>
    </tr>
  </tbody>
</table>
