# Overview

The Branding Video Standout is a hybrid format that bridges awareness and commerce, a storytelling ad that still drives measurable action.

# Response Keys

The following is a description of the different keys used to render ads.

## In the Rendering Section

<table>
  <thead>
    <tr>
      <th>Key</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>branding&#95;image</code></td>
      <td>Image file.<br />Required: <code>true</code>;<br />Width: <code>1200 px</code>;<br />Height: <code>1200 px</code>;<br />Max size: <code>307200 bytes</code> (300 KB);<br />Extensions: <code>jpeg</code>, <code>png</code>.</td>
    </tr>
    <tr>
      <td><code>branding&#95;image&#95;alt&#95;text</code></td>
      <td>Alternative text for the branding image.<br />Required: <code>true</code>;<br />Max characters: <code>120</code></td>
    </tr>
    <tr>
      <td><code>background&#95;color</code></td>
      <td>Background color for the full unit.<br />Required: <code>true</code>;<br />Default: <code>#EFEFEF</code>.</td>
    </tr>
    <tr>
      <td><code>border&#95;color</code></td>
      <td>Border color for the full unit.<br />Required: <code>true</code>;<br />Default: <code>#CCCCCC</code>.</td>
    </tr>
    <tr>
      <td><code>messaging&#95;text</code></td>
      <td>Main ad message.<br />Required: <code>true</code>;<br />Max characters: <code>140</code>.</td>
    </tr>
    <tr>
      <td><code>messaging&#95;text&#95;color</code></td>
      <td>Color of the messaging text.<br />Required: <code>true</code>;<br />Default: <code>#000000</code>.</td>
    </tr>
    <tr>
      <td><code>displayed&#95;brand</code></td>
      <td>Brand name displayed with “Sponsored by” tag.<br />Required: <code>true</code>;<br />Max characters: <code>25</code>.</td>
    </tr>
    <tr>
      <td><code>optional&#95;legal&#95;text</code></td>
      <td>Legal text add-on for the video.<br />Max <code>90 characters</code>, including spaces.<br />Desktop: visible on hover of the “i” icon;<br />Mobile/App: always visible below the video.</td>
    </tr>
    <tr>
      <td><code>optional&#95;redirect&#95;target</code></td>
      <td>Only for web.<br />Ignored if no ClickThrough is provided in the VAST.<br />Values: <code>&#95;blank</code> (new tab), <code>&#95;self</code> (same tab).</td>
    </tr>
    <tr>
      <td><code>background&#95;video&#95;alt&#95;text</code></td>
      <td>Video alternative text for accessibility.</td>
    </tr>
    <tr>
      <td><code>color&#95;mode</code></td>
      <td>Preset theme for UI elements. Values: <code>lightcommand</code> (black elements: <code>#000000</code>), <code>darkcommand</code> (white elements: <code>#FFFFFF</code>).</td>
    </tr>
  </tbody>
</table>

  Fields marked as required are required for video integrations.

## Available Options

### Redirection

Redirection is accomplished by clicking on either:

* the **CTA button**
* the **video** (other than the buttons and legal text)
* the **Branding Image**

There are two possible redirection options:

* `_Self` (opens in the same tab),
* `_Blank` (opens in a new tab).

### Redirection URL

All interactive elements—the CTA button, video, and branding image—redirect to **the same destination URL**.

This URL is retrieved from the VAST XML `&lt;ClickThrough&gt;` section. When defined, this field applies uniformly to all three assets to ensure consistent redirection behavior.

***

### Color Mode

![21479bc0e02a98ae514da5becab4461fbe538a970563480a7c2fc49d67a245b3 Color_mode](/images/retailer-integration/docs/21479bc0e02a98ae514da5becab4461fbe538a970563480a7c2fc49d67a245b3-Color_mode.png)

The advertiser can choose between two preset color modes to ensure the best contrast between the buttons and the video. Buttons and options appear while hovering over the **video**.

#### Light mode

* Background color: `#FFFFFF` 80%
* Icon color: `#6A6A6A`
* Legal text color: `#000000`

#### Dark mode

* Background color: `6A6A6A` 80%
* Icon color: `#FFFFFF`
* Legal text color: `#FFFFFF`

***

## Technical Details

### Global

* The border should be fixed to `1px` and should surround the video creative and the product zone.
* Depending on placement size, the format dimensions may vary. However, the video ratio will remain constant at 16:9.
