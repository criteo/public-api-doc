# Introduction

The publisher (retailer) will most likely already have Google Publisher Tag created and embedded in their website. 

This section is a brief introduction to how GPTs are created and how they may need to be modified.

***

GPTs are typically created by running through the “Tags” wizards for an Ad unit.  Recall that a GPT is just a “physical” manifestation of the logical Ad Unit concept.

# GPT Modification

In order to populate the Macros, the Google Publisher Tag will need to be modified.  The `.setTargeting` invocations in the example below show an example with hard-coded values.  In practice, the `rvid` and `kwd` values, which were mapped to the `retailer-visitor-id` and `context` query parameters for Delivery in the Creative tag, would be populated by request-level variables.

```javascript
googletag.cmd.push(function() {  
    googletag.defineSlot('/2729856/BEN_TEST', [300, 600], 'div-gpt-ad-1718994066072-0').addService(googletag.pubads())  
    .setTargeting("pa", "viewBasket_API_desktop")  
    .setTargeting("pl", "viewBasket_API_desktop-InGrid")  
    .setTargeting("rvid","12345")  
    .setTargeting("kwd","pompeii3")  
    googletag.pubads().enableSingleRequest();
```

The example above demonstrates the modification of a GPT that was auto-generated from the Ad Unit.  Because the GPT is implemented as a JavaScript library, it is also possible for the Retailer to have a more advanced configuration for their GPTs.

## How to know what the request-level variables are?

The retailer is likely already passing them to GAM via their existing tag.  

To find them, examine the request made to GAM:

1. Reload the retailer’s page with Chrome Developer Tools open.
2. Filter by “gam” in the Network tab
3. On the “Payload” tab of this request, find the `prev_scp` and/or `cust_params` query string parameters.

In the following screenshot was taken from [https://www.bestbuy.com](https://www.bestbuy.com):

</Frame>

And after navigating to [https://www.bestbuy.com/site/home-audio-systems/speakers/abcat0205000.c?id=abcat0205000](https://www.bestbuy.com/site/home-audio-systems/speakers/abcat0205000.c?id=abcat0205000) by browsing categories, we see this:
