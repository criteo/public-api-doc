## Response keys

| Field                               | Description                                                                                                                                           |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| `background_image`                  | URL of Background image file for desktop, mobile & app                                                                                                |
| `desktop_background_image_alt_text` | Alternative text for background image file for desktop, mobile & app                                                                                  |
| `redirect_url`                      | Web URL that the user should be redirected to                                                                                                         |
| `redirect_url_app (1)`              | App deeplink that the user should be redirected to                                                                                                    |
| `redirect_target`                   | Redirection target, can have the following values:- “\_blank”: redirects to a new tab
- “\_self”: redirects in the same tab (only applicable for web) |

***

## Specifications

* **Leaderboard**: This format is only displayable on desktop. The format won't be displayed under `728px`.
* **Medium Rectangle**: This format can be displayed on all environment (desktop, mobile web, and app).
* **Skyscraper**: This format is only displayable on desktop.

***

## Asset sizes

* **Leaderboard**: `728x90px`
* **Medium rectangle**: `300x250px`
* **Skyscraper**: `160x600px`
