# Introduction

Sponsored products are native ads that appear directly within a retailer’s product listings or search results, blending seamlessly with the site's content.

***

# Response description

The response of the API call will give you all the information needed to render the ads and track them.

For this reason, the important data are the `productId` in the `products` list to know which product should be displayed, and the different beacons to be able to measure ad performance.

You can find more information about the beacons in the next [chapter](https://developers.criteo.com/retail-media-delivery/docs/retail-media-beacons) and a list of all possible attributes for Sponsored products below.

***

## Main object

| Attributes   | Format                        | Description                                                                                                               |
| ------------ | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------- |
| `status`     | `OK` or `Error`               | Alerts to whether the API call succeeded or failed                                                                        |
| `errors`     | Array of strings              | Only appears if the `status` = `Error`. It will explain the reason behind the error status                                |
| `placements` | Array of objects: `placement` | Lists all placements available for the specific page-id the API is being called for (example: `viewCategory_API_desktop`) |

***

## Placement

| Attributes                    | Format                     | Description                                                                               |
| ----------------------------- | -------------------------- | ----------------------------------------------------------------------------------------- |
| `format`                      | Array of objects: `format` | Lists all formats present for the placement                                               |
| `rendering`                   | Empty string               | Internal use                                                                              |
| `OnClickBeacon`               | Empty string               | Not in use for Sponsored Products                                                         |
| `OnLoadBeacon` `OnViewBeacon` | String containing a URL    | You can find more information on the beacons in the next [chapter](#tracking-ad-activity) |

***

## Formats

This object contains only two members.

The `format` is the type of format. For Sponsored Products, it will always be `sponsored_products`. `products` will then be a list of products that has to be rendered.

***

## Products

| Attributes                                                                                            | Format                          | Description                                                                                                                      |
| ----------------------------------------------------------------------------------------------------- | ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `ProductId`                                                                                           | String                          | Product ID of the product that matches what Criteo has in the product feed                                                       |
| `ProductName` `Image` `Rating` `Price` `ComparePrice` `Shipping` `PromoText` `ParentSKU`              | String                          | Values that are in the feed, describing the `productId` above                                                                    |
| `RenderingAttributes`                                                                                 | String containing a JSON object | Values in the feed that are not standard, but are needed to render the ad. If you need these, contact your Criteo representative |
| `ClientAdvertiserId`                                                                                  | String                          | Internal ID                                                                                                                      |
| `AdvertiserId`                                                                                        | String                          | Criteo ID of the advertiser who sent the ad                                                                                      |
| `ProductPage` `OnLoadBeacon` `OnViewBeacon` `OnClickBeacon` `OnBasketChangeBeacon` `OnWishlistBeacon` | String containing a URL         | More information on the beacons can be found in the next [chapter](#tracking-ad-activity)                                        |

***
