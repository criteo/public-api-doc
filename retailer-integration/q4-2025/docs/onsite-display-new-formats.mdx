## Overview

<Warning>
  As of November 2025, Criteo has improved and revamped **five formats for Onsite Display**. Those formats are simpler to implement and fit better with current standards and technical constraints. We strongly recommended using the new formats for any new implementation, as legacy formats will be progressively phased out.
</Warning>

### What changed?

The five following formats have been redesigned and renamed: `Flagship`, `Showcase`, `Butterfly`, `Digital Shelf Talker` and `Display Panel`.

The mobile-specific formats `Branded Header` and `Interactive Header` were integrated into the new formats.

The`IAB Banners` remain unchanged.

| Format Type      | Legacy Formats              | New Formats                         |
| ---------------- | --------------------------- | ----------------------------------- |
| Commerce Display | Flagship                    | **Commerce Display Spotlight**      |
| Commerce Display | Showcase                    | **Commerce Display Grid Shelf**     |
| Commerce Display | Butterfly                   | **Commerce Display Grid Duet**      |
| Commerce Display | Branded Header (Mobile)     | Now integrated into the new formats |
| Commerce Display | Interactive Header (Mobile) | Now integrated into the new formats |
| Display Banners  | Digital Shelf Talker        | **Branding Display Grid Solo**      |
| Display Banners  | Display Panel               | **Branding Display Spotlight Solo** |
| Display Banners  | IAB Banners                 | IAB Banners (unchanged)             |

<Info>
  The documentation for the legacy formats remains available [here](/retailer-integration/docs/commerce-display-formats#/).
</Info>
