## Retrieving all Advertisers

You can use this endpoint to fetch a list of all the advertisers in your account.

![](/images/marketing-solutions/v2025.10/docs/8a6bfd9-get.png)

```
https://api.criteo.com/2025-10/advertisers/me
```

 The endpoint will return an array of the **IDs** and **names** of all advertisers you've been granted access to.

**Response example:**

<CodeGroup>
  ```bash bash
  {
      "data": [
          {
             "type": "advertiser",
             "id" "1234",
             "attributes": {
                 "advertiserName": "Advertiser A"
             },
             "meta": null
          },
          {
             "type": "advertiser",
             "id" "5678",
             "attributes": {
                 "advertiserName": "Advertiser B"
             },
             "meta": null
          }
      ],
      "errors": [],
      "warnings": []
  }
  ```
</CodeGroup>

<Info>
  **Reference**

  You can find this endpoint in [our reference section](https://developers.criteo.com/marketing-solutions/reference/listadvertisers) as well.
</Info>
