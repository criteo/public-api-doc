1. Marketing Solutions Endpoints

This section documents the Commerce Grid API

## Introduction

Criteo's Commerce Grid Audience Segments API allows you to manage your audience segments by creating, deleting, and updating their data.

<Info>
  You can find more information about Commerce Grid on [our Help Center](https://docs.commercegrid.criteo.com/kb/en).
</Info>

***

## Authentication

You can authenticate to the Commerce Grid API using the endpoint documented [here](https://developers.criteo.com/marketing-solutions/docs/authentication).

***

## Audience Segments Endpoints

You can manage available information, including the name and the description of the audience segments. For audience segments of type contact list, you can manage the users included on them, and retrieve their statistics.
