Our [client libraries](/marketing-solutions/docs/api-client-libraries) for the Criteo API are now available. This transition SDK will help you migrate from MAPI to Criteo API and include endpoints in the latest stable version 2021-04, as well as /legacy endpoints.

[Client libraries](/marketing-solutions/docs/api-client-libraries) - (SDKs) are available in the following languages:

* Python
* PHP
* Java
* Node.js - (Unofficial)

Client libraries are an optional developer tool to help you set up the Criteo API. They can be used to reduce the time to integrate and the amount of code you need to write. Client libraries can also help you with troubleshooting, should you encounter any issues.

As a reminder, MAPI and PAPI will be deprecated on June 15th, 2021. The SDKs can also help you with accelerating your migration.

We will support this SDK until the end of 2021, which is when we will decommission the /legacy endpoints. After that date, we will only support our official versioned client libraries available in Q3 2021 and released on a quarterly basis. Swapping from transition SDK to future SDKs will be seamless for developers, requiring only a change of the package name for the library.

If you use a client library to support your use of the Criteo API, you can find more information [here](/marketing-solutions/docs/api-client-libraries).
