<Info>
  Version **2025.07** of the Criteo API is live as of **July 22nd, 2025** and will be supported until **July 28th, 2026**.  A new Postman collection for 2025.07 is also available in the [Criteo Postman workspace](https://www.postman.com/realcriteo/workspace/criteo/overview).
</Info>

<Info>
  **New SDK Release**

  A new SDK version is available for the `2025.07` release.
</Info>

This version introduces new capabilities to the Criteo API. T

The changes of versions `2025.07` are:

* a new **base URL**,
* a new endpoint in the Preview version for [Real Time Statistics](/marketing-solutions/v2025.10/docs/real-time-statistics)

***

# New Base URL

With the introduction of this new version, the base URL will now change from **the old base URL**:

```text URL
https://api.criteo.com/2025-04/{endpoint}
```

to the **new base URL**:

```text URL
https://api.criteo.com/2025-07/{endpoint}
```
