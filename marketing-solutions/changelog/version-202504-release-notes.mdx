<Info>
  Version **2025.04** of the Criteo API is live as of **April 22nd, 2025** and will be supported until **April 28th, 2026**.  A new Postman collection for 2025.04 is also available in the [Criteo Postman workspace](https://www.postman.com/realcriteo/workspace/criteo/overview).
</Info>

<Info>
  **New SDK Release**

  A new SDK version will be available for the `2025.04` release.
</Info>

This version introduces new capabilities to Criteo API and changes to existing endpoints. This page aims at listing all changes that happened endpoint per endpoint. Only endpoints with changes are listed here.

The changes of versions `2025.04` are:

* a new **base URL**,
* updated **Terms & Conditions**
* updates to the **AdSets endpoints**

<br />

# New Base URL

With the introduction of this new version, the base URL will now change from **the old base URL**:

```text URL
https://api.criteo.com/2025-01/{endpoint}
```

to the **new base URL**:

```text URL
https://api.criteo.com/2025-04/{endpoint}
```

***

# What's New

## Updated Terms & Conditions

We've updated our Terms & Conditions which are available in this portal, and/or when creating a new organization in the API Partner Portal. They focus on the below key areas:

* **Agreement Scope:** The terms apply to the use of Criteo's APIs and are complementary to any main agreements with Criteo or its affiliates.
* **Acceptance:** Users must accept these terms to access and use the APIs, and corporate entities must ensure authorized personnel accept on their behalf.
* **Definitions:** Important terms such as "Affiliate," "API Terms," "Applicable Laws," "Criteo Materials," and "Data Protection Laws" are clearly defined.
* **Compliance:** Users must comply with technical requirements and specifications provided by Criteo.
* **Supplemental Terms:** Depending on API usage, additional terms and conditions may be required.

<Info>
  You can find the new Terms & Conditions [here](/marketing-solutions/docs/criteo-api-terms-and-conditions).
</Info>

## Updated Adset Endpoints

In version `2025.04`, we've introduced new `attributionConfiguration` field to [AdSets](/marketing-solutions/docs/ad-set) endpoints. This is a new feature where advertisers can choose which attribution method to use for their sales campaigns. For more details and for how to enable, please see our help center article [here](https://help.criteo.com/kb/guide/en/connections-7mH0yNkkc4/Steps/2890417)
