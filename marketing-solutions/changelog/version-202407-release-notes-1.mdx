We are pleased to announce the release of Version 2024.10 of the Criteo API. This is a stable API version that will be fully supported until November 2025. Please ensure you review the updates and adjust your integrations accordingly. As always, our updated client libraries (Python, Java, and PHP) and Postman collections are available to help you get started.

## What's New

There are no new endpoints or changes to Version 2024.10.

Thank you for your continued partnership and commitment to using the Criteo API.
