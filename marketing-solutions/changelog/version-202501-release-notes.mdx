<Info>
  Version **2025.01** of the Criteo API is live as of **January 21st, 2025**. This is a stable API version that will be fully supported until **January 27th, 2026**.  Please ensure you review the updates and adjust your integrations accordingly.  A new Postman collection for 2025.01 is available in the [Criteo Postman workspace](https://www.postman.com/realcriteo/workspace/criteo/overview).
</Info>

<Info>
  **New SDK Release**

  A new SDK version will be available for the 2025.01 release available in our [client libraries](/marketing-solutions/docs/api-client-libraries).
</Info>

<br />

This version introduces new capabilities to Criteo API and changes to existing endpoints. This page aims at listing all changes that happened endpoint per endpoint. There were no changes to endpoints in this version.

The changes of versions 2025.01 are:

* a new **base URL**

<br />

# New Base URL

With the introduction of this new version, the base URL will now change from **the old base URL**:

```text URL
https://api.criteo.com/2024-10/{endpoint}
```

to the **new base URL**:

```text URL
https://api.criteo.com/2025-01/{endpoint}
```
