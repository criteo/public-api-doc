1. Marketing Solutions Endpoints

## **Audience API Endpoint**

Criteo's **Audience API** allows you to create and manage Audiences and Audience Segments.

Audiences are built up of Audience Segments, which are groupings of users from either Criteo-powered data, lookalike models, or an advertiser’s own data. Audiences can then be linked to one or more Ad Sets once built. Once linked, Audiences drive to whom the Ad Set should deliver (e.g. people interested in shoes).

## **Use Cases**

Here's a list of the use cases of this API. For a more detailed explanation, refer to the specific pages.

### Create audiences to define the people an ad set should target

If you want to create one or more audiences by combining audience segments that define in detail the type of people you would like to target.

### Manage audiences

If you want to retrieve and update one or more audiences you own, to keep them up-to-date.

### Delete audiences

If you want to delete one or more audiences that are not relevant to you anymore and are not being used.

### Create audience segments to define the people to target

If you want to create one or more audience segments to target people based on their shopping intent, location, and similarity to your existing website visitors.

### Create Contact List segments to target existing customers

If you want to create one or more Contact List audience segments to target existing customers based on their emails or other identifiers.

### Manage audience segments

If you want to retrieve and update one or more audience segments you own, to keep them up-to-date.

### Delete audience segments

If you want to delete one or more audience segments that are not relevant to you anymore and are not being used.

### Retrieve the size of one or more existing audiences or audience segments

If you want to retrieve the size of one or more audiences or audience segments you own to know approximately how many users are targeted.

### Estimate the size of a non-existing audience or audience segment

If you want to estimate the size of an audience or audience segment before creating it, in order to configure it to target the number of users you desire.
