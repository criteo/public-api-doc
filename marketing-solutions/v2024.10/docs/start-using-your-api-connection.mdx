**Start using the Criteo API endpoints**

* You will see the accounts that have accepted your consent request using the endpoint /advertisers/me (you can try it in the [API Reference](/marketing-solutions/v2024.10/api-reference/advertiser/portfolio-get)).
* You can then use the domains in your scope for those advertisers/brands/publishers and start using Criteo APIs! 🎉
