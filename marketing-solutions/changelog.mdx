<Update label="January 2025" description="Version 2025.01 by j.jaffe@criteo.com">
    <Info>
        Version **2025.01** of the Criteo API is live as of **January 21st, 2025**. This is a stable API version that will be fully supported until **January 27th, 2026**. Please ensure you review the updates and adjust your integrations accordingly. A new Postman collection for 2025.01 is available in the [Criteo Postman workspace](https://www.postman.com/realcriteo/workspace/criteo/overview).
    </Info>

<Update label="October 2024" description="Version 2024.10 by j.jaffe@criteo.com">
   We are pleased to announce the release of Version 2024.10 of the Criteo API. This is a stable API version that will be fully supported until November 2025. Please ensure you review the updates and adjust your integrations accordingly. As always, our updated client libraries (Python, Java, and PHP) and Postman collections are available to help you get started.
