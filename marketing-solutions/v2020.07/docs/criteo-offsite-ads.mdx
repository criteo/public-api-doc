## Criteo Offsite Ads endpoint

```http
https://api.criteo.com/legacy/offsite-ads/
```

## Introduction

The Criteo Offsite Ads (COA) API allows you to manage your COA campaigns.   **Criteo OffSite Ads require "Campaign" domain permission.** (see [step 3](/marketing-solutions/docs/onboarding-checklist#3-define-your-app-scope))

This API allows you to :

* Retrieve the list of **Sellers** associated with your campaign
* Set the **bids** of your **Seller Campaigns**
* Configure the **budgets** for your **Seller Campaigns**
* Review the performance of your COA campaigns by querying their delivery statistics
