## Endpoints

| Verb | Endpoint | Description |
|---|---|---|
| `GET` | `/marketing-solutions/ads/{ad-id}/product-boost` | Fetch all boosting associations and configurations for an ad. |
| `GET` | `/marketing-solutions/ads/{ad-id}/product-boost/{product-set-id}` | Fetch the boosting association and configuration for a specific product set. |
| `POST` | `/marketing-solutions/ads/{ad-id}/product-boost/{product-set-id}` | Create or update a product boosting configuration. |
| `DELETE` | `/marketing-solutions/ads/{ad-id}/product-boost/{product-set-id}` | Delete a boosting association and configuration. |
| `GET` | `/marketing-solutions/dataset/{dataset-id}/product-boost` | Fetch boosting associations and configurations for a given partner. |

## Get All Boosting Configurations for an Ad

<EndpointBadge method="get">
```http
https://api.criteo.com/{version}/marketing-solutions/ads/{ad-id}/product-boost
```
</EndpointBadge>

Returns all product set boosting associations and their configurations for the specified ad.

**Path parameters**

| Parameter | Type | Description |
|---|---|---|
| `ad-id` | string | ID of the ad. |

**Sample response**

```json
{
  "data": [
    {
      "type": "BoostedAdProductSet",
      "attributes": {
        "adId": "12345",
        "productSetId": "67890",
        "boostingFactor": 1.5,
        "modificationDate": "2026-07-01T10:00:00Z"
      }
    }
  ],
  "errors": [],
  "warnings": []
}
```

## Create or Update a Boosting Configuration

<EndpointBadge method="post">
```http
https://api.criteo.com/{version}/marketing-solutions/ads/{ad-id}/product-boost/{product-set-id}
```
</EndpointBadge>

Creates a new boosting configuration or updates an existing one. Returns `201` on creation and `200` on update.

**Sample request**

```json
{
  "data": {
    "type": "BoostingConfigurationRequest",
    "attributes": {
      "boostingFactor": 1.5
    }
  }
}
```

**Sample response**

```json
{
  "data": {
    "type": "BoostedAdProductSet",
    "attributes": {

      "boostingFactor": 1.5,

}
```
