## **Advertisers API Endpoint**

```text
https://api.criteo.com/preview/advertisers
```

## **Introduction**

Criteo's **Advertisers API** will allow you to **create** a new advertiser and retrieve a list of valid **industries** with which to classify a new advertiser.

You can also easily retrieve the **names** and **IDs** of all advertisers you've been granted access to.

   

<Warning>
  **Known Limitations**

  Advertiser creation is not yet available. Currently, you can only retrieve your portfolio of advertisers.\
  Full functionality will be added in future releases.
</Warning>
