## Introduction

The Recommendations Search API from Criteo allows you to retrieve personalized product recommendations based on specific parameters enhancing your platform's user experience. By utilizing this API, you can deliver targeted content to your users and improve engagement.

[Product Recommendations](https://www.criteo.com/technology/product-recommendations/) use our deep understanding of each shopper’s individual habits to serve up personalized items tailored to your goals.

<Warning>
  To fully use our app you need to have read access to the Product Recommendation domain. To set up it please reach out to Criteo Support ([criteotechnicalpartnerships@criteo.com](mailto:criteotechnicalpartnerships@criteo.com)).
</Warning>

## Recommendation Search API Endpoint

```text POST
 https://api.as.criteo.com/preview/recommendation/search
```
