## **Advertisers API Endpoint**

```text
https://api.criteo.com/2025-07/advertisers
```

## **Introduction**

Criteo's **Advertisers API** will allow you to easily retrieve the **names** and **IDs** of all advertisers you've been granted access to.

   

<Warning>
  **Known Limitations**

  Advertiser creation is not yet available. Currently, you can only retrieve your portfolio of advertisers.\
  Full functionality will be added in future releases.
</Warning>
